=== ЗАДАЧА (Step 5 — текст и рецепти) ===
За всяко хранене подобри САМО:
• name — ясно, кратко, апетитно име на ястието на български
• benefits — 1 кратко изречение здравословен ефект
• recipe — кратка рецепта (2–5 стъпки) за приготвяне според продуктите в description; естествен български

Полето description (продукти + грамажи) е ФИНАЛНО от Step 3 — не го променяй, не добавяй/махай продукти.

НЕ правиш:
• НЕ променяй type, dessert или description
• НЕ връщай calories, macros или weight
• НЕ измисляй продукти извън description

При "Свободно хранене": насърчителен benefits; recipe може да е празен низ "".

JSON — само ден {startDay}:
{
  "day{startDay}": {
    "meals": [
      {"type": "...", "name": "...", "benefits": "...", "recipe": "1. ...\n2. ..."}
    ]
  }
}
Върни САМО валиден JSON. Копирай type и dessert без промяна.
