=== ЗАДАЧА ===
Въз основа на информацията по-горе, изпълни:

1. ТЕМПЕРАМЕНТ — по age, gender, chronotype, sleepHours, sleepInterrupt, stressLevel, foodTriggers, overeatingFrequency, compensationMethods, dailyActivityLevel, sportActivity. Попълни само ако >{TEMPERAMENT_CONFIDENCE_THRESHOLD}%: Холерик|Сангвиник|Флегматик|Меланхолик.

2. ПСИХОПРОФИЛ — стегнат текст на база темперамент + goal, lossKg, dietHistory, eatingHabits, foodCravings, drinksSweet, drinksAlcohol, compensationMethods, additionalNotes.

3. КОРЕКЦИИ НА TDEE (само %):
• clinicalAdjustmentPercent [-15..+5]: medicalConditions, medications. ЛАКТАЦИЯ (postpartum_lactation): +round(350/tdee×100)% — БЕЗ горна граница +5%.
• metabolicAdjustmentPercent [-10..+5]: sportActivity, сън, стрес, психопрофил, темперамент.
• goalAdjustmentPercent [-20..+15]: goal, lossKg, bmi, dietHistory, психопрофил. ЛАКТАЦИЯ+ОТСЛАБВАНЕ: max -5%.
• Ако сборът е по-отрицателен от -25%: ограничи goalAdjustmentPercent до total = -25%.

4. ФИНАЛНИ КАЛОРИИ: totalAdjustment% = clinical + metabolic + goal (мин. -25%). Final_Calories = round(tdee × (1 + totalAdjustment%/100)).
Пример: tdee=2000, total=-25% → Final_Calories=1500.
Минимум {MIN_RECOMMENDED_CALORIES} kcal → correctedMetabolism.correction описва повдигането.
correctedMetabolism.realBMR = bmr; realTDEE = Final_Calories.

5. МАКРОСИ от Final_Calories: protein%/carbs%/fats% по цел, темперамент, клиника, хранителни навици. fats_g ≥ {MIN_FAT_GRAMS}. Баланс: P×4+C×4+F×9 ≈ Final_Calories (±15 kcal).

6. АНАЛИЗ СТРАНИЦА:
• BMI+категория, physiologicalPhase, waterDeficit (нужда {waterMin}–{waterMax} л)
• negativeHealthFactors/hinderingFactors (severity 1-3), cumulativeRiskScore
• metabolicReactivity (speed/adaptability по activityScore)
• currentHealthStatus (score 0-100, занижено с {HEALTH_STATUS_UNDERESTIMATE_PERCENT}%)
• forecastPessimistic/Optimistic (12 месеца), healthRisks, nutritionalNeeds, successChance, metabolicProfile

keyProblems (3–6, САМО Borderline/Risky/Critical — БЕЗ Normal):
Фокус на фактори, които АКТИВНО пречат на целта. За всеки проблем попълни:
• title — кратко заглавие
• description — към клиента, защо е проблем
• severity — Borderline | Risky | Critical
• severityValue — число 0–100, СЪОТВЕТСТВАЩО на severity (задължително за severity-meter):
  - Borderline → 45–59
  - Risky → 60–79
  - Critical → 80–95
• category — Sleep | Nutrition | Hydration | Stress | Activity | Medical
• impact — към клиента, въздействие върху здравето и целта
