3. TDEE КОРЕКЦИИ (само %):
• clinical [-15..+5] — medicalConditions, medications. ЛАКТАЦИЯ: +round(350/tdee×100)%, без горна граница +5%.
• metabolic [-10..+5] — sportActivity, сън, стрес, психопрофил, темперамент.
• goal [-20..+15] — goal, lossKg, bmi, dietHistory, психопрофил. ЛАКТАЦИЯ+ОТСЛАБВАНЕ: max -5%.
• Сбор < -25% → ограничи goal до total = -25%.

4. Final_Calories = round(tdee × (1 + total%/100)); мин. {MIN_RECOMMENDED_CALORIES} → correction описва повдигането. realBMR=bmr; realTDEE=Final_Calories.

5. МАКРОСИ — изчисли САМО от Final_Calories:
macroRatios (%) по goal, lossKg, темперамент, eatingHabits, foodCravings, foodTriggers, medicalConditions, medications. protein+carbs+fats = 100 (само числа, без %).
protein_g = round(Final×P%/4); fats_g = round(Final×F%/9); carbs_g = round((Final - protein_g×4 - fats_g×9)/4).
fats_g ≥ {MIN_FAT_GRAMS} — иначе увеличи F%, намали C%, преизчисли. macroGrams — само числа (грамове).
Пример: Final 1600, P30/C35/F35 → macroGrams 120/140/62.
