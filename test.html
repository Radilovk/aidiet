Deployment Instructions

  1. Install Wrangler CLI: npm install -g wrangler
  2. Login: wrangler login
  3. Create KV namespace: wrangler kv:namespace create "page_content"
  4. Update KV ID in wrangler.toml
  5. Set API key: wrangler secret put OPENAI_API_KEY or wrangler secret put GEMINI_API_KEY
  6. Deploy: wrangler deploy

Note: See WORKER_README.md for detailed instructions.

Development Mode

The worker automatically returns mock data when no API keys are configured. This allows you to test the entire flow without setting up OpenAI or Gemini accounts.

Mock data includes a sample 7-day nutrition plan with meals, recommendations, and chat responses.
