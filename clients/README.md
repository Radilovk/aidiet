# Clients Folder

Тази папка съхранява данни от клиенти, попълнили Въпросник 2.0.

## Как работи

1. **Попълване на въпросника**: Клиентът попълва questionnaire2.html
2. **Изтегляне**: След завършване, клиентът натиска бутона "Изтегли данните"
3. **Съхранение**: Изтегленият JSON файл се записва ръчно в тази папка

## Структура на данните

Всеки клиентски файл съдържа:
- **id**: Уникален идентификатор на клиента (формат: `client_{timestamp}_{random}`)
- **timestamp**: Дата и час на подаване (ISO 8601 формат)
- **answers**: Отговорите от въпросника (всички 31+ полета)
- **files**: Прикачени файлове (ако има такива) в base64 формат

## Пример на име на файл

```
client_1771087769249_p4o9ji.json
```

## Бележки

- Файловете се генерират автоматично от questionnaire2.html
- Данните се изтеглят директно от браузъра на клиента
- Файловете в тази папка не се следят от Git (.gitignore)
- Използвайте този формат за архивиране и обработка на клиентски данни
- Вижте EXAMPLE_client_data.json за примерна структура
