PWA Icons Required
==================

Please create the following icon files:

1. icon-192x192.png (192x192 pixels)
2. icon-512x512.png (512x512 pixels)

Design requirements:
- Background: #10b981 (green)
- Logo/Symbol: White or light colored
- Simple, recognizable icon representing NutriPlan
- PNG format with transparent or solid background
- Suggested design: Apple icon or "NP" letters

You can use any graphic design tool (Figma, Photoshop, Canva, etc.) or 
online icon generators to create these icons.

For testing purposes, you can also use favicon generators like:
- https://favicon.io/
- https://realfavicongenerator.net/
